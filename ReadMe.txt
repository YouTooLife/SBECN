Для работы программ на компьютере требуется:
.NET Framework 4.5.2
Java Runtime Environment 1.6.0

Запустите ValidatorSBECN.exe из папки Validator
-Составьте список избирателей
-Составьте список кандидатов
-Настрока сервера:
*Введите адрес сервера регистратора (можно оставить localhost если работа будет происходить внутри локальной сети) и наждмите на кнопку (>>)
*Из выпадающего списка выберите любой пункт
(по этому адресу будут подключаться клиенты)
*Введите любой свободный порт (кроме 1013)
*Наждмите кнопку (Начать голосование)

Запустите CounterSBECN.exe из папки Counter
-Настрока сервера счетчика:
*Введите адрес сервера счетчика (можно оставить 127.0.0.1 если работа будет происходить внутри локальной сети) и наждмите на кнопку (>>)
*Из выпадающего списка выберите любой пункт
-Настрока сервера регистратора:
*Введите адрес сервера регистратора и наждмите на кнопку (>>)
*Из выпадающего списка выберите пункт, адрес каторого равен адресу сервера регистратора
*Введите порт сервера регистратора
-Нажмите кнопку (Подключиться)

Запустите ClientSBECN.exe из папки Client
-Настрока сервера регистратора:
*Введите адрес сервера регистратора и наждмите на кнопку (>>)
*Из выпадающего списка выберите пункт, адрес каторого равен адресу сервера регистратора
*Введите порт сервера регистратора
-Введите любой свободный порт для создания сервера клиента (кроме 1013)
-Нажмите кнопку (Подключиться)

::Работа программ проверялась только внутри локальной сети!
